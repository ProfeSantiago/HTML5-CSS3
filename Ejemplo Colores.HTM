Los colores en Microsoft Internet Explorer 4

Para especificar un color en HTML, utilice las ternas RGB. Por ejemplo, si desea que una página tenga el color de fondo AliceBlue, utilice la siguiente etiqueta:

<BODY BGCOLOR=#F0F8FF>

Utilice el número decimal en los controles ActiveX y miniaplicaciones Java que requieran números decimales.
